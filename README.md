# Level Up Gamer - V0.7 (Geolocation)

## 📝 Descripción
Mejora de la experiencia de compra con gestión de direcciones y mapas.

## 🚀 Novedades
- **Google Maps**: Integración de mapas para visualización de puntos de entrega.
- **Modelo Direcciones**: Tabla y modelo para múltiples direcciones por usuario.
- **UI Checkout**: Primera fase del flujo de selección de dirección.

## 🛠️ Actualización
1. Ejecutar `DB/update_v0.7.sql`.
2. Requiere Play Services habilitado en el emulador/dispositivo.

---
*Nota: Es necesario contar con una API Key de Google Cloud para el renderizado del mapa.*
