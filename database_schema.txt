
# Esquema de la Base de Datos

Este archivo documenta el esquema de la base de datos de la aplicación Level Up Gamer. 
La base de datos está implementada con Room Persistence Library.

---

## Tabla: `clientes`

Representa a los clientes de la aplicación.

| Columna          | Tipo    | Descripción                               |
|------------------|---------|-------------------------------------------|
| `Id`             | `Int`   | Clave primaria, autoincremental.          |
| `usuario_id`     | `Int`   | ID del usuario asociado al cliente.       |
| `nombre_completo`| `String`| Nombre completo del cliente.              |
| `telefono`       | `String`| Número de teléfono del cliente.           |
| `direccion`      | `String`| Dirección del cliente.                    |

---

## Tabla: `detalle_pedido`

Representa el detalle de un pedido específico.

| Columna       | Tipo    | Descripción                               |
|---------------|---------|-------------------------------------------|
| `Id`          | `Int`   | Clave primaria, autoincremental.          |
| `Pedido_Id`   | `Int`   | ID del pedido al que pertenece el detalle.|
| `Producto_Id` | `Int`   | ID del producto en el detalle.            |
| `Cantidad`    | `Int`   | Cantidad del producto en el detalle.      |
| `Subtotal`    | `Double`| Subtotal del detalle del pedido.          |

---

## Tabla: `pedidos`

Representa los pedidos realizados por los clientes.

| Columna       | Tipo    | Descripción                               |
|---------------|---------|-------------------------------------------|
| `Id`          | `Int`   | Clave primaria, autoincremental.          |
| `ClienteId`   | `Int`   | ID del cliente que realizó el pedido.     |
| `FechaPedido` | `String`| Fecha en que se realizó el pedido.        |
| `Estado`      | `String`| Estado actual del pedido.                 |
| `Total`       | `Double`| Total del pedido.                         |

---

## Tabla: `Producto`

Representa los productos disponibles en la tienda.

| Columna         | Tipo    | Descripción                               |
|-----------------|---------|-------------------------------------------|
| `id`            | `Int`   | Clave primaria, autoincremental.          |
| `codigo`        | `Double`| Código único del producto.                |
| `nombre`        | `String`| Nombre del producto.                      |
| `categoria`     | `String`| Categoría a la que pertenece el producto. |
| `descripcion`   | `String`| Descripción del producto.                 |
| `precio`        | `Double`| Precio del producto.                      |
| `cantidad`      | `Int`   | Cantidad disponible en stock.             |
| `imagenUrl`     | `String`| URL de la imagen del producto.            |
| `imagenLocal`   | `String`| Nombre del recurso de imagen local.       |

---

## Tabla: `reseñas`

Representa las reseñas de los productos hechas por los clientes.

| Columna       | Tipo    | Descripción                               |
|---------------|---------|-------------------------------------------|
| `id`          | `Int`   | Clave primaria, autoincremental.          |
| `cliente_id`  | `Int`   | ID del cliente que escribió la reseña.    |
| `producto_id` | `Int`   | ID del producto reseñado.                 |
| `calificacion`| `Int`   | Calificación dada al producto.            |
| `comentario`  | `String`| Comentario de la reseña.                  |
| `fecha`       | `String`| Fecha en que se escribió la reseña.       |

---

## Tabla: `tipo_usuario`

Representa los diferentes tipos de usuarios en el sistema.

| Columna       | Tipo    | Descripción                               |
|---------------|---------|-------------------------------------------|
| `id`          | `Int`   | Clave primaria, autoincremental.          |
| `nombre`      | `String`| Nombre del tipo de usuario.               |
| `descripcion` | `String`| Descripción del tipo de usuario.          |

---

## Tabla: `usuario`

Representa a los usuarios del sistema.

| Columna         | Tipo     | Descripción                               |
|-----------------|----------|-------------------------------------------|
| `id`            | `Int`    | Clave primaria, autoincremental.          |
| `nombre`        | `String` | Nombre de usuario.                        |
| `contrasena`    | `String` | Contraseña del usuario.                   |
| `email`         | `String` | Correo electrónico del usuario.           |
| `tipo_usuario_id`| `Int`    | ID del tipo de usuario asociado.          |
| `activo`        | `Boolean`| Indica si el usuario está activo.         |
| `fecha_creacion`| `String` | Fecha de creación del usuario.            |
